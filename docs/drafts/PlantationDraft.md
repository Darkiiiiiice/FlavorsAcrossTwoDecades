# 星夜小馆后院属性系统设计

后院是星夜小馆的“绿洲”，既是食材来源，也是盼盼放松心情的地方。这里可以种植蔬菜、香料和花卉，收获的作物用于烹饪、装饰或赠送邻居。后院还与盼盼的旅行系统联动——旅行可能带回新品种的种子。玩家通过盼盼远程管理后院，从一片荒芜逐步开垦成生机勃勃的小菜园。

---

## 一、后院等级

后院拥有一个综合等级，1-5级，反映整体开垦和升级程度。等级提升会解锁新地块、新作物类型，并影响作物生长速度。

| 等级 | 名称 | 解锁条件 | 效果 |
|------|------|----------|------|
| 1级 | 荒芜初垦 | 初始状态 | 仅有1块小菜地可用，基础作物可种 |
| 2级 | 菜畦成行 | 开垦2块菜地，修复灌溉基础 | 解锁香料种植，生长速度+10% |
| 3级 | 花果满园 | 开垦3块菜地，搭建工具房 | 解锁花卉种植，可吸引蜜蜂（提高产量） |
| 4级 | 四季常青 | 开垦4块菜地，升级灌溉系统 | 解锁反季节种植，可种特殊作物 |
| 5级 | 星夜花园 | 开垦全部5块菜地，搭建温室 | 解锁珍稀作物，可与旅行系统联动获得异星种子 |

## 二、区域划分与子设施

后院分为多个区域，每个区域有独立的状态。

### 2.1 菜地
| 属性 | 说明 |
|------|------|
| **总地块数** | 5块（初始解锁1块，后续每升一级解锁1块） |
| **每块面积** | 可同时种植1种作物，每块有独立生长进度 |
| **初始状态** | 土壤板结，需翻土才能种植 |
| **土壤肥力** | 每块地有肥力值0-100，种植消耗肥力，施肥可恢复。肥力影响作物生长速度和最终产量。 |
| **升级路径** | 每块地可单独升级：<br>1级：普通土壤（基础）<br>2级：改良土壤（施肥+松土，消耗材料）<br>3级：肥沃黑土（需购买或事件获得） |

### 2.2 灌溉系统
| 属性 | 说明 |
|------|------|
| **初始状态** | 无，盼盼需手动浇水（占用时间） |
| **最高等级** | 3级（智能滴灌系统） |
| **影响** | 浇水效率、作物生长速度、盼盼劳作时间 |
| **升级路径** | 1级→2级：安装简易水管，¥800，盼盼可自己铺设<br>2级→3级：升级为自动滴灌，¥3000，需专业安装，解放盼盼双手 |

### 2.3 工具房
| 属性 | 说明 |
|------|------|
| **初始状态** | 破旧小屋，存放基础农具 |
| **最高等级** | 3级（设备齐全的工具间） |
| **影响** | 可存放种子、肥料、农具，影响盼盼工作效率 |
| **升级路径** | 1级→2级：整理修缮，¥500，盼盼整理，增加储物架<br>2级→3级：扩建，¥2000，可存放更多工具，解锁工坊联动 |

### 2.4 堆肥区
| 属性 | 说明 |
|------|------|
| **初始状态** | 无 |
| **最高等级** | 2级（可制作有机肥） |
| **影响** | 可利用厨余垃圾制作肥料，节省成本 |
| **升级路径** | 1级→2级：搭建堆肥箱，¥300，盼盼学习堆肥方法（需事件触发） |

### 2.5 温室（可选）
| 属性 | 说明 |
|------|------|
| **解锁条件** | 后院4级 + 特殊材料（旅行带回） |
| **影响** | 可反季节种植，保护特殊作物 |
| **等级** | 仅1级（基本温室） |

## 三、种植系统

### 3.1 作物类型
| 类别 | 示例 | 用途 | 生长时间（现实） |
|------|------|------|------------------|
| **蔬菜** | 番茄、辣椒、小葱、豆角 | 烹饪食材 | 2-4天 |
| **香料** | 薄荷、罗勒、香菜 | 提升菜品品质 | 3-5天 |
| **花卉** | 茉莉、月季、雏菊 | 装饰、赠送顾客 | 4-7天 |
| **特殊作物** | 祖父留下的神秘种子、旅行带回的异星种子 | 解锁特殊事件 | 不定 |

### 3.2 生长阶段
每块地的作物经历四个阶段：
1. **播种期**：盼盼播种后，进入此阶段，持续1天（现实）
2. **发芽期**：需浇水，持续1-2天
3. **生长期**：需定期施肥、除草，持续2-3天
4. **成熟期**：可收获，持续2天，之后开始枯萎

### 3.3 影响因素
- **土壤肥力**：肥力高则生长加速，产量提高
- **浇水**：灌溉系统等级高则自动浇水，否则需盼盼手动
- **施肥**：可使用堆肥区制作的肥料或购买肥料
- **季节**：某些作物只在特定季节生长（温室可反季节）
- **盼盼技能**：盼盼的“种植技能”（需设计，可与旅行技能合并？建议单独设“园艺技能”）影响照料效率和产量

### 3.4 产量
每块地成熟时可收获的作物数量受肥力、盼盼技能影响。基础产量为5份，最高可达15份。

## 四、收获与用途

### 4.1 收获管理
作物成熟时，盼盼会汇报，玩家可选择：
- **立即收获**：盼盼收获后存入库存
- **稍后收获**：作物仍在田中，但盼盼会提醒
- **留种**：部分作物可留种（需特殊技能），下次种植不需购买种子

### 4.2 用途
| 用途 | 说明 | 收益 |
|------|------|------|
| **烹饪** | 直接用于厨房菜品，提升新鲜度加成 | 菜品品质+0.5~1星 |
| **赠送邻居** | 送给老街坊，增进邻里关系 | 互助值+，可能解锁事件 |
| **装饰** | 花卉可插在餐厅花瓶，提升氛围 | 氛围指数+2~5 |
| **售卖** | 可在邻里市场出售（需玩家决定） | 获得资金 |
| **制作** | 部分作物可用于工坊制作（如干花书签） | 解锁新物品 |

### 4.3 新鲜度
收获的作物有新鲜度属性（100%），随时间下降，冰箱可减缓下降。新鲜度低于50%时，不适合赠送或装饰，但可用于烹饪（品质加成减半）。

## 五、与盼盼属性关联

| 盼盼属性 | 后院相关影响 |
|----------|--------------|
| **园艺技能**（新增） | 影响播种成功率、作物生长速度、产量、病虫害处理能力。可通过种植经验升级。 |
| **维修技能** | 影响灌溉系统、工具房的维修效率 |
| **旅行技能** | 高等级旅行可能带回稀有种子或种植技术 |
| **情绪** | 心情好时，盼盼更愿意主动照料后院；疲惫时可能疏忽浇水 |
| **信任度** | 高信任度时，盼盼会主动提出种植计划优化建议 |
| **能量** | 后院劳作消耗能量（浇水、施肥、收获等），需注意盼盼电量 |

## 七、总结

后院属性系统为星夜小馆提供了自给自足的食材来源和额外的社交元素。玩家通过盼盼管理后院，既能提升餐厅菜品品质，又能加深与老街邻居的联系。后院的成长与盼盼的园艺技能、旅行收获相互促进，成为游戏经营循环中充满田园趣味的一环。
