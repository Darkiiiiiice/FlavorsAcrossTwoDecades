# 星夜小馆顾客系统设计

顾客是星夜小馆的“常驻民”，每一位老顾客都承载着老街的记忆，也与祖父有着千丝万缕的联系。玩家通过盼盼的间接观察了解他们，并通过事件选择影响他们的故事走向。顾客系统是记忆碎片的重要来源，也是小馆“人情味”的核心体现。

---

## 一、顾客基础属性

每位顾客包含以下属性：

| 属性 | 说明 |
|------|------|
| **姓名** | 如“李大爷”、“王奶奶”、“老周” |
| **头像** | 像素风格图像，由盼盼摄像头捕捉（初期模糊，后期清晰） |
| **身份** | 老街上的角色，如退休工人、作家、花店老板等 |
| **出现规律** | 每日/每周固定时间段出现（如李大爷每天早上7点来吃早饭） |
| **常点菜品** | 喜欢的菜品，可能影响满意度 |
| **性格** | 如开朗、寡言、热心、挑剔等，影响事件选项 |
| **好感度** | 0-100，影响是否解锁个人故事和记忆碎片 |
| **状态** | 正常 / 生病 / 外出 / 有心事等，盼盼会察觉 |
| **与祖父关系** | 如老友、邻居、受恩惠者等，决定记忆碎片内容 |

---

## 二、顾客类型

### 2.1 固定老顾客
每天都来或每周固定几天来的顾客，是小馆的“班底”。他们的故事会随着好感度提升逐步解锁。

| 姓名 | 身份 | 出现时间 | 常点菜品 | 与祖父关系 |
|------|------|----------|----------|------------|
| 李大爷 | 退休机械师 | 每天早餐时段 | 豆浆油条 | 祖父老友 |
| 王奶奶 | 花店老板娘 | 每天下午 | 菊花茶 | 邻居，常与祖父交换种子 |
| 老周 | 自由撰稿人 | 每天下午至傍晚 | 咖啡、阳春面 | 祖父曾鼓励他写作 |
| 小美 | 年轻白领 | 每周二、四晚餐 | 糖醋排骨 | 祖父资助过她上学 |

### 2.2 流动顾客
偶尔出现的新面孔，可能成为潜在老顾客，或带来特别事件。

| 类型 | 说明 |
|------|------|
| 旅行者 | 路过老街，被招牌吸引 |
| 美食博主 | 可能撰写推荐，影响口碑 |
| 老街新移民 | 刚搬来，尝试附近餐馆 |
| 神秘访客 | 可能携带祖父的秘密 |

### 2.3 特殊顾客
只在特定事件或条件下出现的顾客，与结局相关。

| 姓名 | 出现条件 | 作用 |
|------|----------|------|
| 父亲（真相） | 记忆碎片集齐后 | 揭示真相结局 |
| 祖父幻影 | 高信任度特殊事件 | 传递最后信息 |

## 三、顾客好感度系统

### 3.1 好感度获取
- **菜品合口味**：每次点常点菜品且制作品质高，好感度+1~3
- **特殊关怀**：事件中选择帮助或送小礼物，好感度+5~15
- **记住偏好**：盼盼能记住并主动满足顾客习惯，好感度缓慢提升

### 3.2 好感度影响
| 好感度区间 | 效果 |
|------------|------|
| 0-20 | 普通顾客，偶尔来 |
| 21-40 | 熟客，出现频率增加，盼盼开始记录喜好 |
| 41-60 | 忠实顾客，会主动介绍新客人，可能触发初级事件 |
| 61-80 | 老街朋友，愿意分享个人故事，解锁部分记忆碎片 |
| 81-100 | 亲人般的存在，会参与小馆的重大事件，解锁全部个人记忆碎片 |

---

## 四、顾客事件系统

### 4.1 事件触发
- **基于好感度**：达到特定阈值触发
- **基于时间**：生日、节日等特殊日子
- **基于小馆状态**：新菜研发成功、设施升级等
- **随机**：日常小插曲

### 4.2 事件类型
| 类型 | 示例 | 影响 |
|------|------|------|
| **个人故事** | 李大爷回忆与祖父钓鱼往事 | 解锁记忆碎片，好感度提升 |
| **求助** | 王奶奶的猫丢了 | 选择是否帮忙，影响邻里关系 |
| **分享** | 老周新书出版，送来签名本 | 解锁装饰品，氛围提升 |
| **危机** | 小美失业，情绪低落 | 选择是否安慰，影响小美后续 |
| **惊喜** | 顾客帮忙宣传，带来新客流 | 口碑临时提升 |

### 4.3 事件决策
- 盼盼向玩家请示，玩家通过延迟通信做出选择。
- 不同选择影响：
  - 顾客好感度变化
  - 盼盼性格轴偏移
  - 记忆碎片解锁路径
  - 小馆口碑

## 五、顾客与记忆碎片

每位固定老顾客都与祖父有过去的交集，他们的故事构成记忆碎片的重要部分。

| 顾客 | 记忆碎片标题 | 解锁条件 | 内容 |
|------|--------------|----------|------|
| 李大爷 | “钓鱼的早晨” | 好感度60+，触发聊天 | 祖父和李大爷年轻时一起钓鱼，祖父用钓到的鱼做了一道菜 |
| 王奶奶 | “花与种子” | 后院种植成功，赠送花卉 | 祖父教王奶奶种花，交换过神秘种子 |
| 老周 | “写作的鼓励” | 好感度70+，老周新书发布 | 祖父曾鼓励老周坚持写作，说“小馆永远给你留个位置” |
| 小美 | “上学的路费” | 好感度80+，小美升职 | 祖父曾匿名资助小美上学，小美后来发现是祖父 |

## 六、顾客图鉴

游戏内设“顾客图鉴”，记录所有已解锁的顾客及其故事进度。图鉴包含：
- 顾客头像（清晰度随好感度提升）
- 基本信息（姓名、身份、出现规律）
- 个人故事解锁进度
- 好感度
- 记忆碎片列表（已解锁/待解锁）

## 八、与盼盼属性关联

| 盼盼属性 | 顾客相关影响 |
|----------|--------------|
| **社交技能** | 影响盼盼观察顾客情绪、记录偏好的准确性；高技能能更早察觉顾客心事。 |
| **记忆容量** | 影响盼盼能记住多少顾客的细节（如生日、喜好）。 |
| **信任度** | 高信任度时，盼盼会主动提醒玩家“今天是李大爷生日”或“王奶奶看起来不开心”。 |
| **情绪** | 盼盼情绪好时，与顾客互动更愉快，间接提升顾客满意度。 |
| **旅行技能** | 旅行带回的纪念品可赠送顾客，大幅提升好感度。 |

## 九、顾客状态报告示例

盼盼每日简报会包含顾客动态，例如：

> **【星夜小馆·顾客动态】**
> - 今日客流量：23人，其中老顾客12人
> - 李大爷：7:15到店，点了豆浆油条，心情不错，聊起年轻时钓鱼的事。
> - 王奶奶：15:30到店，带来一束茉莉，说放在餐厅里香。她提到后院的花该施肥了。
> - 老周：16:00到店，在角落写作，点了两杯咖啡，看起来有些焦虑（可能赶稿）。
> - 新顾客：一对年轻情侣路过，点了番茄鸡蛋面，评价“味道像家里做的”。
> - 特殊提醒：明天是李大爷生日，建议准备长寿面。

## 十、总结

顾客系统将星夜小馆的人情味具象化。每一位老顾客都是祖父记忆的见证者，也是玩家与地球连接的情感纽带。通过盼盼的观察和事件选择，玩家逐渐拼凑出祖父完整的人生图景，也让小馆重新成为老街的精神家园。顾客的好感度、个人故事和记忆碎片，共同构成游戏叙事的重要支柱。
