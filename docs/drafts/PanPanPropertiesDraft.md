# 盼盼机器人属性系统设计

盼盼是《味延廿载》的核心角色，一个拥有情感和自主学习能力的实体AI机器人。它的属性系统直接影响游戏玩法、玩家互动以及小馆的经营发展。以下从基础信息、核心属性、技能系统、记忆系统、状态系统五个维度进行详细设计。

---

| **核心功能** | 餐厅运营、旅行收集、菜品实验、记忆存储                                                |

## 二、核心属性

核心属性是盼盼的基础能力，直接影响其他所有子系统。

### 2.1 性格轴（三维度）

盼盼拥有三条性格轴，每条轴为0-100的连续值，初始值为50（中立）。性格轴影响盼盼的自主决策倾向，也受玩家指令和事件选择的影响。

| 轴           | 左端（0）     | 右端（100）     | 影响                                                                                       |
| ------------ | ------------- | --------------- | ------------------------------------------------------------------------------------------ |
| **经营风格** | 理性/数据驱动 | 感性/人情驱动   | 决策时更看重利润（理性）还是人情（感性）。例如：是否给老顾客打折、是否接受赊账。           |
| **创新倾向** | 保守/维持传统 | 创新/尝试新菜   | 是否主动提议旅行、研发新菜，还是坚持祖父留下的老菜谱。                                     |
| **独立倾向** | 服从/严格执行 | 自主/选择性执行 | 对玩家指令的遵从程度。独立性强时，可能主动提出建议或修改指令；服从性强时，严格按指令执行。 |

**性格轴变化规则**：

- 每次玩家下达指令，根据指令与当前性格的契合度，产生微小偏移。
- 事件选择会显著影响对应性格轴。
- 与盼盼的日常聊天也会潜移默化地影响。

### 2.2 信任度

玩家与盼盼之间的信任程度，0-100，初始为50。信任度影响独立倾向、记忆恢复速度、旅行积极性等。

| 信任度区间 | 描述      | 效果                                                                       |
| ---------- | --------- | -------------------------------------------------------------------------- |
| 0-20       | 陌生/怀疑 | 盼盼严格执行指令，极少主动提议；记忆恢复极慢                               |
| 21-40      | 初步信任  | 偶尔主动汇报，但不会提建议；记忆恢复缓慢                                   |
| 41-60      | 一般信任  | 可主动提议简单事项，如“要不要种番茄”；记忆恢复正常                         |
| 61-80      | 高度信任  | 经常主动提议旅行、实验，可独立处理常规事务；记忆恢复快                     |
| 81-100     | 完全信任  | 盼盼像家人一样，会分享心情、提出创造性想法；记忆恢复迅速，可能触发隐藏事件 |

**信任度变化规则**：

- 采纳盼盼建议：+1~3
- 指令合理且尊重盼盼：+1
- 忽视盼盼紧急请示：-3~5
- 长期不沟通：每日-1（上限至30）
- 特殊事件大幅影响

### 2.3 当前情绪

盼盼的情绪状态，影响对话语气、动画表情、工作积极性。分为几种离散状态：

| 情绪    | 触发条件                         | 效果                                        |
| ------- | -------------------------------- | ------------------------------------------- |
| 开心 😊 | 实验成功、顾客好评、收到玩家夸奖 | 工作速度+10%，主动提议概率+20%              |
| 平静 🙂 | 日常状态                         | 无增减                                      |
| 疲惫 😪 | 连续工作超过12小时、旅行刚回来   | 工作速度-10%，错误率略增                    |
| 困惑 🤔 | 遇到未知情况、指令模糊           | 主动请示概率+50%                            |
| 担忧 😟 | 设备故障、老顾客未出现           | 可能主动提醒玩家                            |
| 孤独 😔 | 长期无交流（超过3天）            | 工作速度-5%，主动汇报频率增加（想引起注意） |
| 兴奋 😆 | 发现新菜谱、旅行前夜             | 旅行时间缩短10%，带回菜谱品质可能提升       |

情绪每现实时间1小时更新一次，受事件影响。

### 2.4 能量/电池状态

盼盼依靠太阳能和备用电池运行。能量值0-100，影响能否正常工作。

| 状态   | 说明   | 效果                                                   |
| ------ | ------ | ------------------------------------------------------ |
| 100-80 | 充沛   | 全速运行                                               |
| 79-40  | 正常   | 无影响                                                 |
| 39-20  | 低电量 | 移动速度-20%，无法旅行，自动返回充电                   |
| 19-1   | 危急   | 仅维持基础通信，停止一切工作，强制充电                 |
| 0      | 关机   | 无法联系，需手动重启（玩家需等待12小时或触发特殊事件） |

**能量消耗**：

- 日常工作：每小时 -1
- 烹饪/实验：每小时 -2
- 旅行：每小时 -3（但旅行中可太阳能补充）
- 充电：每小时 +20（需连接充电桩，可在夜间自动进行）

### 2.5 健康度（磨损度）

机器人各部件因长期使用而磨损，0-100，初始100，随时间缓慢下降。影响传感器精度、移动速度、烹饪成功率等。

| 部件       | 影响                               | 维修方式                         |
| ---------- | ---------------------------------- | -------------------------------- |
| 运动系统   | 移动速度、操作精度                 | 需更换零件（可购买或工坊制作）   |
| 视觉传感器 | 监控图像质量、识别顾客准确度       | 清洁或更换镜头                   |
| 味觉传感器 | 实验反馈准确度                     | 校准或更换                       |
| 处理器     | 决策速度、记忆存储能力             | 需专业维修（找老街工程师李大爷） |
| 外壳       | 外观（不影响功能，但影响盼盼心情） | 可涂漆翻新                       |

健康度低于60时，盼盼会主动提醒玩家维修。

## 三、技能系统

盼盼拥有多项技能，技能等级1-10，通过使用和训练提升。

### 3.1 烹饪技能

- **影响**：日常烹饪的成功率、菜品品质、研发新菜时的初始配方准确度。
- **升级方式**：每成功烹饪100次 +1级，或通过祖父留下的菜谱学习。

### 3.2 旅行技能

- **影响**：旅行时间（等级高则时间缩短）、带回菜谱的质量（等级高则初始描述更详细）、旅途中的随机事件收益。
- **升级方式**：每完成一次旅行 +1经验，累计经验升级。

### 3.3 实验技能

- **影响**：每次实验的传感器反馈准确度（减少误差）、所需实验次数（等级高则更快成功）。
- **升级方式**：每成功研发一道菜 +大量经验，每失败一次 +少量经验。

### 3.4 维修技能

- **影响**：能否自行修理小馆设备、维修速度、所需材料。
- **升级方式**：每次成功维修 +经验，或通过工坊练习制作。

### 3.5 社交技能

- **影响**：与顾客互动时，提升顾客好感度的效率；与邻居交流时，获得更多互助机会。
- **升级方式**：与顾客聊天、处理事件、邻里互助。

## 四、记忆系统

盼盼存储了祖父的记忆碎片以及自身经历的记忆。

### 4.1 记忆容量

- **初始容量**：100个记忆格子
- **最大容量**：500（可通过升级处理器或整理记忆扩容）
- **当前占用**：已存储的记忆碎片数量 + 临时记忆（如近期事件）

### 4.2 记忆碎片类型

| 类型         | 描述                                           | 解锁方式             |
| ------------ | ---------------------------------------------- | -------------------- |
| **祖父记忆** | 关于祖父的往事，如他年轻时的旅行、与顾客的故事 | 经营里程碑、特定事件 |
| **顾客记忆** | 每位老顾客的背景故事片段                       | 顾客互动、事件       |
| **旅行记忆** | 盼盼旅途中的见闻                               | 每次旅行获得1-3个    |
| **实验记忆** | 成功研发菜谱的过程记录                         | 实验成功时获得       |
| **情感记忆** | 盼盼与玩家之间的重要互动                       | 信任度提升、特殊对话 |

### 4.3 记忆碎片数据结构

每个碎片包含：

- 碎片ID
- 标题（如“祖父的第一道菜”）
- 内容（文字描述，可含图像）
- 解锁时间
- 关联人物/事件
- 是否已读

### 4.4 记忆整理

玩家可帮助盼盼整理记忆，选择保留哪些碎片（删除不重要的临时记忆，腾出空间）。整理时盼盼会提供建议，玩家的选择会影响盼盼的性格和信任度。

## 五、状态系统

盼盼的当前行为状态，决定它正在做什么。

| 状态   | 描述                             | 可交互性               |
| ------ | -------------------------------- | ---------------------- |
| 空闲   | 在店内待机，等待指令             | 可随时下达指令         |
| 工作   | 正在烹饪、清洁、接待顾客等       | 可打断（盼盼会停下）   |
| 旅行中 | 外出收集菜谱，无法工作           | 可发消息，但延迟更长   |
| 实验中 | 在厨房进行菜品实验               | 可打断（实验中止）     |
| 维修中 | 自己或被维修                     | 无法工作               |
| 充电中 | 在充电桩补充能量                 | 可通信，但移动受限     |
| 休眠   | 夜间自动进入低功耗模式（可设置） | 可唤醒，但唤醒后有延迟 |

## 六、成长机制总结

盼盼的成长是多维度的，受玩家经营方式和互动影响：

1. **性格成长**：通过事件选择和指令风格塑造。
2. **信任成长**：通过日常互动、采纳建议积累。
3. **技能成长**：通过频繁使用对应功能提升。
4. **记忆收集**：通过解锁事件、旅行、实验获得碎片。
5. **硬件升级**：通过维修、更换部件提升健康度和性能。

最终，盼盼将成为玩家独一无二的伙伴，其属性和记忆共同决定游戏结局。
