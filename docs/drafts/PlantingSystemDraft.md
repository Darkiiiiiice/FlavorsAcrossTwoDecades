# 星夜小馆种植系统设计

种植是星夜小馆后院的核心功能，也是盼盼亲近自然、收获新鲜食材的重要途径。从荒芜的土地到生机盎然的菜园，玩家通过盼盼远程管理每一块菜地，种植蔬菜、香料和花卉。种植系统与厨房、工坊、邻里和旅行系统深度联动，为小馆提供新鲜的食材和温情的社交礼物。

---

## 一、种植概述

后院共有**5块菜地**，初始仅解锁1块，随着后院等级提升逐步解锁更多。每块菜地可同时种植1种作物，拥有独立的土壤肥力、生长进度和状态。种植受现实时间影响，作物生长需要数小时到数天不等，玩家需通过盼盼定期照料。

---

## 二、作物种类与属性

### 2.1 作物分类

| 类别 | 特点 | 示例 | 主要用途 |
|------|------|------|----------|
| **蔬菜** | 生长周期短，基础食材 | 番茄、辣椒、小葱、豆角 | 烹饪核心食材 |
| **香料** | 提升菜品品质，生长周期中等 | 薄荷、罗勒、香菜、迷迭香 | 烹饪调味，工坊制作 |
| **花卉** | 生长周期较长，观赏性强 | 茉莉、月季、雏菊、桂花 | 装饰餐厅，赠送顾客 |
| **特殊作物** | 稀有，与事件/旅行相关 | 祖父留下的神秘种子、异星植物 | 特殊菜谱，记忆碎片触发 |

### 2.2 作物属性
每类/每种作物包含以下属性：

| 属性 | 说明 |
|------|------|
| **名称** | 如“番茄” |
| **类型** | 蔬菜/香料/花卉/特殊 |
| **生长周期** | 播种到成熟所需现实时间（小时/天） |
| **成熟季节** | 适宜生长的季节（春/夏/秋/冬/全年），温室可反季节 |
| **理想温度** | 影响生长速度（由盼盼传感器监控） |
| **需水量** | 每天需要浇水次数（受天气和土壤影响） |
| **需肥量** | 生长过程中对肥力的消耗速度 |
| **产量** | 成熟时可收获的数量（受肥力、盼盼技能影响） |
| **稀有度** | 普通/稀有/传说（影响种子获取难度） |
| **用途** | 烹饪/装饰/赠送/制作 |
| **关联菜谱** | 可用于哪些菜品的制作 |

## 三、种子系统

### 3.1 种子来源
- **初始赠送**：祖父留下的基础种子（番茄、小葱、薄荷）
- **采购**：可在虚拟市场购买常见种子（消耗资金）
- **邻里互助**：王奶奶等邻居会赠送稀有花种
- **旅行带回**：盼盼旅行可能带回异域种子（如四川辣椒、普罗旺斯香草）
- **事件获得**：特殊事件解锁隐藏种子（如“祖父的玫瑰”）
- **留种**：部分作物成熟后可留种（需盼盼园艺技能达到一定等级）

### 3.2 种子属性
每颗种子记录：
- **作物种类**
- **品质**（影响最终作物品质上限，1-5星）
- **产地**（如“成都花椒种子”、“王奶奶的茉莉”）
- **是否可留种**

### 3.3 种子存储
种子存放在后院工具房的种子盒中，玩家可通过盼盼查看库存。种子无保质期，但稀有种子建议尽早种植（避免丢失）。

## 四、种植流程

### 4.1 播种
1. **选择菜地**：玩家在“后院Tab”选择一块空闲菜地。
2. **选择种子**：从库存中选择要种植的种子。
3. **确认播种**：盼盼自动进行播种操作，消耗种子，菜地进入“播种期”。

### 4.2 生长阶段
每块菜地的作物经历以下阶段：

| 阶段 | 状态 | 持续时间（示例） | 需要操作 |
|------|------|------------------|----------|
| **播种期** | 种子刚播下，未发芽 | 4-8小时 | 无 |
| **发芽期** | 幼苗破土而出 | 8-12小时 | 适量浇水 |
| **生长期** | 植株快速成长 | 1-2天 | 定期浇水、施肥 |
| **成熟期** | 作物可收获 | 2-3天（之后逐渐枯萎） | 及时收获 |
| **枯萎期** | 错过收获，作物衰败 | 1天后消失 | 清理后重新播种 |

各阶段持续时间受作物种类、季节、土壤、照料等因素影响。

### 4.3 日常照料
- **浇水**：盼盼可根据灌溉系统自动浇水，或手动指令浇水。缺水会导致生长停滞甚至枯萎。
- **施肥**：使用堆肥区制作的肥料或购买肥料，提升土壤肥力，促进生长。
- **除草/除虫**：发现杂草或害虫时，盼盼可手动清除（需玩家指令或盼盼自动发现）。
- **状态监控**：盼盼每日汇报每块菜地的状态，包括水分、肥力、是否有病虫害。

### 4.4 收获
- 作物成熟时，盼盼主动推送消息提醒玩家。
- 玩家可选择“立即收获”或“稍后”（作物在成熟期可停留数天，之后进入枯萎期）。
- 收获时，根据作物产量获得相应数量的“收成”（如番茄×5）。
- 收获后菜地变回空闲，可重新播种。

## 五、生长影响因素

### 5.1 土壤肥力
- 每块菜地有独立的肥力值（0-100），初始50。
- 种植消耗肥力（每生长周期消耗10-30不等）。
- 施肥可提升肥力（堆肥+5，高级肥料+10）。
- 休耕可缓慢恢复肥力（每天+2）。
- 肥力影响生长速度和最终产量。

### 5.2 浇水
- 每块菜地有水分值（0-100），盼盼可查看。
- 灌溉系统自动维持水分，否则盼盼需手动浇水（消耗时间和能量）。
- 缺水时生长停滞，水分过低作物枯萎。

### 5.3 季节与温度
- 游戏内模拟地球季节（与现实同步）。季节影响作物生长速度：
  - 适宜季节：生长速度×1.2
  - 非适宜季节：生长速度×0.8
  - 冬季无温室时，部分作物无法生长
- 温室可反季节种植，但建造温室需后院4级及特殊材料。

### 5.4 盼盼园艺技能
- 盼盼拥有**园艺技能**（1-10级），通过种植经验提升。
- 高园艺技能影响：
  - 播种时选择更优的种子（自动筛选）
  - 提高浇水施肥效率
  - 更早发现病虫害
  - 增加作物产量（每级+2%）
  - 解锁留种能力（4级可留普通种子，7级可留稀有种子）

## 六、病虫害与灾害

### 6.1 病虫害类型
| 类型 | 表现 | 处理方法 |
|------|------|----------|
| **蚜虫** | 叶片卷曲，生长停滞 | 盼盼可手动喷洒药水（需购买） |
| **霉菌** | 叶片出现霉斑，可能传染 | 需改善通风，喷洒杀菌剂 |
| **杂草** | 争夺养分，生长变慢 | 盼盼手工拔除 |
| **鸟雀** | 啄食果实 | 可搭建稻草人（工坊制作） |

### 6.2 触发条件
- 连续阴雨、高温高湿易引发霉菌
- 肥力过高或不平衡易招虫
- 缺乏照料时杂草生长

### 6.3 灾害
- **极端天气**：霜冻、冰雹、干旱等（根据现实天气或随机事件），可损坏作物。
- **预防措施**：温室、遮阳网、灌溉系统升级可降低灾害损失。

## 七、收获用途

### 7.1 烹饪
- 新鲜收获的蔬菜/香料可直接用于厨房，提升菜品品质（新鲜度加成）。
- 某些菜谱需要特定作物（如“后院番茄蛋汤”）。

### 7.2 赠送
- 花卉和部分蔬菜可赠送顾客或邻居，提升好感度。
- 例如：送王奶奶茉莉，送李大爷番茄。

### 7.3 装饰
- 花卉可插入餐厅花瓶，提升氛围指数（每束花+2~5氛围，持续数天）。
- 干花可在工坊制作成装饰品。

### 7.4 制作
- 部分作物可用于工坊制作（如香料制香皂、花瓣制书签）。

### 7.5 售卖
- 可在邻里市场出售多余作物，获得资金（价格低于采购价，但纯利润）。

## 八、与盼盼属性关联

| 盼盼属性 | 种植相关影响 |
|----------|--------------|
| **园艺技能** | 影响播种、照料效率、产量、留种能力 |
| **维修技能** | 影响灌溉系统、温室设施的维修 |
| **能量** | 照料作物消耗能量（浇水、施肥、除虫等） |
| **情绪** | 心情好时更主动照料；疲惫时可能疏忽 |
| **信任度** | 高信任度时，盼盼会提出种植计划优化建议（如“该种番茄了”） |

## 九、与其他系统的联动

| 系统 | 联动方式 |
|------|----------|
| **厨房** | 提供新鲜食材，提升菜品品质 |
| **工坊** | 提供制作材料（花卉、香料），同时工坊可制作温室、稻草人等 |
| **邻里** | 赠送作物提升邻里关系，可能获得回馈（种子、肥料） |
| **旅行** | 旅行带回新种子，丰富种植品种 |
| **顾客** | 某些顾客对后院作物有偏好（如王奶奶喜欢花卉），赠送解锁故事 |
| **记忆碎片** | 特殊作物（如祖父留下的玫瑰）可解锁记忆碎片 |

## 十二、总结

种植系统为星夜小馆提供了源源不断的新鲜食材和温情互动。玩家通过盼盼远程照料每一株作物，感受从种子到收获的喜悦。与厨房、工坊、邻里系统的深度联动，让后院不仅是食材来源，更是小馆人情味的重要延伸。随着园艺技能的提升和后院的升级，玩家将解锁更多稀有作物，让星夜小馆的味道更加丰富多元。
