# 星夜小馆菜谱与实验系统设计

菜谱是星夜小馆的灵魂，每一道菜都承载着祖父的记忆或盼盼旅途的见闻。游戏中的菜谱分为两类：**祖父传承菜谱**（初始拥有部分，但可能模糊或损坏）和**旅行实验菜谱**（盼盼带回模糊描述，通过实验研发）。菜谱系统与厨房、盼盼属性、旅行系统深度绑定。

---

## 一、菜谱基础属性

每道菜谱包含以下属性：

| 属性 | 说明 |
|------|------|
| **菜谱ID** | 唯一标识 |
| **菜名** | 如“麻婆豆腐”、“祖父红烧肉” |
| **菜系** | 川菜、粤菜、家常菜等，影响顾客偏好 |
| **来源** | 祖父传承 / 旅行获得 / 事件获得 |
| **状态** | 模糊（未研发） / 研发中 / 已研发 |
| **描述** | 简短介绍，可能包含故事背景 |
| **食材列表** | 研发成功后精确列出每种食材的用量 |
| **制作时间** | 基础制作时间（分钟） |
| **基础售价** | 建议售价，玩家可调整 |
| **基础品质** | 1-5星，决定顾客满意度基础 |
| **稀有度** | 普通 / 稀有 / 传说（影响事件触发概率） |

---

## 二、模糊菜谱

盼盼旅行归来带回的是“模糊菜谱”，只有文字描述和线索。例如：

> **麻婆豆腐（模糊）**
> - 来源：盼盼成都之旅
> - 描述：“麻辣鲜香的川菜，豆腐嫩滑，牛肉末香酥。需要辣椒和花椒，但用量不详。老厨师说最重要的是‘麻辣平衡’。”
> - 主要食材：豆腐、牛肉末、辣椒、花椒、豆瓣酱、蒜苗
> - 线索：可能提示关键比例（如“辣椒和花椒1:1”但不确定）

模糊菜谱无法直接制作，必须经过实验研发。

## 三、实验研发流程

### 3.1 实验启动条件
- 拥有一个模糊菜谱
- 厨房设施健康度均≥70%
- 拥有实验所需的基础食材（至少每种食材有3份以上）
- 盼盼能量≥30

### 3.2 初始配方生成
盼盼根据以下因素生成初始预估配方：
- 模糊菜谱的描述和线索
- 盼盼的**实验技能**（影响初始配方的准确度）
- 是否有类似菜品的研发经验（如已研发川菜，对辣椒用量更有把握）

初始配方包含每种食材的“预估用量”（单位：克或份），并附有“置信度”（0-100%）。置信度影响后续实验的收敛速度。

### 3.3 实验迭代
每次实验盼盼按照当前配方制作一份菜品，然后通过传感器分析，给出反馈。

**传感器反馈类型**：
- **用量偏多**：如“辣椒多了，建议减少5克”
- **用量偏少**：如“花椒少了，建议增加3克”
- **用量合适**：如“盐刚好”
- **其他因素**：如“火候可能偏大”（非用量问题，但影响品质）

传感器反馈的准确度受以下因素影响：
- 盼盼**实验技能**（高技能反馈更精准）
- 厨房设备健康度（尤其灶台、烤箱的温控精度）
- 食材新鲜度（新鲜食材更易判断）

### 3.4 玩家指导
每次实验后，玩家可查看反馈，并下达宏观指导：
- **“按反馈调整”**：盼盼自动根据反馈调整配方（推荐）
- **“这次辣椒多一点”**：玩家手动强调某项调整
- **“保持原样再试一次”**：有时需要确认反馈是否稳定
- **“跳过这次”**：放弃本次实验，不消耗食材

玩家也可设置“自动迭代”模式，让盼盼在信任度高时自行连续实验，但需消耗食材。

### 3.5 配方收敛
每次调整后，配方逐步接近理想值。当所有食材用量连续两次实验反馈均为“合适”时，系统判定该食材达到理想范围。当所有食材都达到理想范围，且额外因素（如火候）也达标，则实验成功。

## 四、成功判定机制

### 4.1 理想用量范围
每种食材都有一个隐藏的“理想用量范围”（如辣椒15-18克）。游戏设计时预先设定每道菜的理想参数。实验过程就是让预估用量落入这个范围。

### 4.2 传感器模型
传感器并非绝对精确，而是带有一定的误差。盼盼的反馈基于当前配方与理想范围的差距，但会受技能和设备影响：
反馈信号 = 真实差距 × (1 - 误差率)
误差率 = (100 - 盼盼实验技能)/200 + (100 - 设备健康度)/200
误差率范围0~0.5。高技能+好设备可使误差率接近0。

例如，真实辣椒用量多了5克，若误差率0.2，则反馈可能提示“多了4克”或“多了6克”。

### 4.3 成功标准
系统内部为每道菜定义一个“综合评分”：
综合评分 = Σ(各食材偏差百分比 × 权重) + 火候评分 + 其他因素
- 各食材偏差百分比：预估用量与理想范围的相对偏差（超出范围的部分）
- 权重：不同食材重要性不同（如辣椒比蒜苗重要）
- 火候评分：基于盼盼烹饪技能和灶台状态的模拟

当综合评分 ≥ 95分（满分100）时，实验成功，菜谱被精确记录。

### 4.4 成功后的菜谱
研发成功后，菜谱变为“已研发”状态，记录精确用量。可加入菜单正常售卖。

---

## 五、影响实验成功的因素

| 因素 | 影响 |
|------|------|
| **盼盼实验技能** | 影响初始配方准确度、反馈误差率、每次迭代的收敛速度 |
| **盼盼烹饪技能** | 影响火候评分，进而影响最终成功率 |
| **厨房设备健康度** | 影响反馈误差，尤其灶台、烤箱的温控精度 |
| **食材新鲜度** | 新鲜食材让反馈更准确，品质更高 |
| **实验次数** | 某些复杂菜谱需要更多迭代，即使技能高也需耐心 |
| **玩家指导** | 合理的宏观指导可加速收敛 |
| **随机因素** | 小概率的“灵感迸发”（幸运事件）可一次成功 |

## 六、菜谱品质与价值

研发成功后，菜谱有基础品质（1-5星），受以下因素影响：
- 菜谱本身的稀有度
- 实验过程中的平均评分（多次实验可优化）
- 盼盼烹饪技能（首次成功时的技能等级）

品质影响：
- 顾客满意度基础值
- 建议售价
- 可能触发特殊事件（如美食评论家到访）

菜谱可重复实验优化，每次实验若品质更高，可更新菜谱的精确配方。

## 八、示例实验日志

> **【实验日志：麻婆豆腐（第3次）】**
> - 当前配方：豆腐300g，牛肉末50g，辣椒15g，花椒8g，豆瓣酱20g，蒜苗20g
> - 传感器反馈：
>   - 辣椒：稍少（建议+2g）
>   - 花椒：合适
>   - 豆瓣酱：稍多（建议-3g）
>   - 火候：稍欠（灶台温度偏低）
> - 盼盼建议：增加辣椒至17g，减少豆瓣酱至17g，注意灶台火力。
> - 玩家选择：采纳建议
> - 下次实验预计食材消耗：豆腐300g，牛肉末50g，辣椒2g，花椒0g，豆瓣酱-3g（实际17g），蒜苗20g

## 九、总结

菜谱与实验系统将旅行收集、厨房经营、盼盼技能融为一体。每一次实验都是科学与艺术的结合，玩家通过远程指导盼盼，逐步还原祖父的味道或创造新的风味。成功的那一刻，不仅是菜品的诞生，更是记忆与情感的延续。
