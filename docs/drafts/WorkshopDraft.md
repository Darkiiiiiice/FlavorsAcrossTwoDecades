# 星夜小馆工坊属性系统设计

工坊是星夜小馆的“百宝箱”，位于后院旁的工具房里。这里存放着各种工具和材料，盼盼可以在这里制作小物件、维修设备、甚至研发新的工具。工坊既是盼盼发挥创造力的空间，也是提升小馆品质、加深邻里关系的重要场所。玩家通过盼盼远程管理工坊，逐步升级设备，解锁更多制作配方。

---

## 一、工坊等级

工坊拥有一个综合等级，1-5级，反映整体设备和功能完善程度。等级提升会解锁新的制作配方、提升制作效率，并影响工坊与其它系统的联动。

| 等级 | 名称 | 解锁条件 | 效果 |
|------|------|----------|------|
| 1级 | 杂物小间 | 初始状态 | 基础工具可用，可制作简单物品 |
| 2级 | 工具小屋 | 整理工具房，添置基础工作台 | 解锁中等复杂度物品，制作速度+15% |
| 3级 | 手艺工坊 | 升级工作台，增加材料架 | 解锁高级物品，可维修复杂设备 |
| 4级 | 创意工间 | 安装专业工具墙，引入电动工具 | 可制作特殊装饰品，与后院联动增强 |
| 5级 | 星夜工坊 | 全部设施顶级，盼盼专属设计台 | 解锁隐藏配方，可制作与结局相关的特殊物品 |

## 二、子设施详细设计

工坊包含多个子设施，每个设施有**等级**（1-5级）和**健康度**（0-100%），健康度随使用下降，需定期维护。设施升级需要消耗资金、材料和时间。

### 2.1 工作台

| 属性 | 说明 |
|------|------|
| **初始状态** | 老旧木桌，台面不平，工具散乱 |
| **最高等级** | 5级（智能升降工作台，带照明和测量尺） |
| **影响** | 制作效率、可制作的物品复杂度、制作成功率 |
| **健康度影响** | 健康度低于50%时，制作时间+30%，失败率+15% |
| **升级路径** | 1级→2级：修整桌面，整理工具，¥300，盼盼自行完成<br>2级→3级：购买新工作台，¥1500，盼盼组装<br>3级→4级：加装台灯、测量工具，¥800，解锁精细制作<br>4级→5级：升级智能工作台，¥3000，需盼盼学习新技能 |

### 2.2 材料架

| 属性 | 说明 |
|------|------|
| **初始状态** | 几个破旧木架，存放少量基础材料 |
| **最高等级** | 4级（分类储物架，带标签和防潮功能） |
| **影响** | 材料存储容量、取用效率、材料保鲜（如木材防潮） |
| **升级路径** | 1级→2级：修理旧架子，¥200，盼盼整理分类<br>2级→3级：购买新金属架，¥1000，盼盼组装<br>3级→4级：加装密封盒、标签系统，¥600，解锁材料分类自动提醒 |

### 2.3 工具墙

| 属性 | 说明 |
|------|------|
| **初始状态** | 墙上几颗钉子，挂着锤子、螺丝刀 |
| **最高等级** | 4级（洞洞板+电动工具专用区） |
| **影响** | 工具取用方便度（影响制作速度），可存放的工具种类 |
| **升级路径** | 1级→2级：安装木板挂架，¥150，盼盼自制<br>2级→3级：购买专业洞洞板，¥500，盼盼安装<br>3级→4级：增加电动工具专用区，需通电源，¥1200 |

### 2.4 维修区

| 属性 | 说明 |
|------|------|
| **初始状态** | 无专门区域，盼盼在院子地上维修 |
| **最高等级** | 3级（带升降台的专用维修位） |
| **影响** | 维修大型设备的能力、维修速度 |
| **升级路径** | 1级→2级：划分区域，铺设垫子，¥200，盼盼整理<br>2级→3级：购置小型升降台，¥2000，可维修更重设备 |

### 2.5 电源与照明

| 属性 | 说明 |
|------|------|
| **初始状态** | 一盏昏暗灯泡，插座不足 |
| **最高等级** | 3级（多插座+明亮LED照明） |
| **影响** | 能否使用电动工具、夜间工作效率 |
| **升级路径** | 1级→2级：更换灯泡，增加插排，¥300，盼盼可做<br>2级→3级：重新布线，¥1500，需电工（可找李大爷） |

## 三、制作系统

### 3.1 可制作物品分类

| 类别 | 示例 | 用途 | 所需主要材料 |
|------|------|------|--------------|
| **日常消耗** | 筷子、杯垫、抹布 | 提升小馆细节，顾客好感度轻微增加 | 木材、布料 |
| **维修工具** | 新把手、螺丝配件 | 用于维修小馆设备 | 金属件、塑料 |
| **装饰物品** | 干花书签、木雕小摆件 | 提升餐厅氛围指数 | 花卉（后院）、木材 |
| **礼品** | 手工香皂、小盆栽 | 赠送顾客或邻居，提升好感度 | 花卉、香料（后院） |
| **特殊物品** | 祖父相框、星夜招牌灯 | 与剧情相关，解锁记忆碎片 | 特殊材料、旅行带回 |

### 3.2 制作流程

1. **选择配方**：玩家在工坊Tab查看已解锁配方，选择要制作的物品。
2. **检查材料**：盼盼自动检查材料库存，若不足则提示玩家采购或从后院获取。
3. **开始制作**：玩家确认后，盼盼开始制作，消耗现实时间（分钟到数小时不等）。
4. **制作进度**：盼盼在制作期间可进行其他工作，但制作进度会占用一定注意力（可设置“专注制作”模式，期间盼盼不执行其他任务）。
5. **制作完成**：盼盼通知玩家，物品加入库存。

### 3.3 制作时间与成功率

- **基础时间**：每个配方有基础制作时间（如筷子30分钟，木雕2小时）。
- **影响因素**：
  - 盼盼的**维修技能**（等级高则时间缩短，成功率提升）
  - 工作台等级（高等级缩短时间）
  - 工具墙等级（高等级提升成功率）
  - 盼盼情绪（开心时成功率+5%）
- **成功率**：即使满足条件，仍有小概率失败（尤其复杂物品）。失败后材料部分损耗，盼盼可尝试重做。

### 3.4 配方解锁

配方通过以下方式解锁：
- **初始自带**：基础物品（筷子、杯垫）
- **工坊升级**：升级到特定等级解锁新配方
- **事件获得**：邻里互助中邻居教给盼盼新配方
- **旅行带回**：盼盼旅行时学到当地手工艺
- **记忆碎片**：祖父留下的手稿中包含特殊配方

## 四、材料管理

### 4.1 材料来源

| 来源 | 说明 |
|------|------|
| **采购** | 盼盼可在虚拟市场购买基础材料（木材、金属件、布料等） |
| **后院产出** | 花卉、香料可用于制作物品 |
| **厨房边角料** | 部分厨余可转化为材料（如果核、蛋壳，需堆肥区处理） |
| **邻里互助** | 邻居可能赠送特殊材料（如老王家的木料、李大爷的旧零件） |
| **旅行带回** | 盼盼旅行可能带回当地特色材料（如异国布料、特殊石头） |
| **拆解回收** | 可拆解旧物品获得部分材料 |

### 4.2 材料存储

- 材料存储在工坊的材料架中，有容量限制（受材料架等级影响）。
- 每种材料有数量上限，超出需扩容或消耗。
- 盼盼会定期盘点材料库存，提醒玩家哪些材料短缺。

### 4.3 材料新鲜度/保质期

部分天然材料（如花卉、木材）有保质期或防潮要求。材料架等级高可延长保存时间。

## 五、与其它子系统的联动

| 子系统 | 联动方式 |
|--------|----------|
| **餐厅** | 工坊制作的装饰品可提升餐厅氛围指数；制作的杯垫、筷子等提升顾客满意度 |
| **厨房** | 工坊可维修厨房设备（如灶台零件）；制作厨具（如新锅柄） |
| **后院** | 后院花卉用于制作干花等物品；工坊可制作花架、温室零件 |
| **盼盼属性** | 维修技能直接影响制作效率和成功率；盼盼心情影响制作品质 |
| **邻里系统** | 制作的礼品可赠送邻居，提升互助值，解锁新事件 |
| **记忆碎片** | 特殊物品的制作可能触发祖父的记忆碎片 |

## 六、与盼盼属性的关联

| 盼盼属性 | 工坊相关影响 |
|----------|--------------|
| **维修技能** | 影响所有制作的成功率、速度、可制作物品复杂度。高技能解锁高级配方。 |
| **创新倾向** | 影响盼盼是否主动提出新物品制作建议；创新倾向高时可能尝试改良配方。 |
| **信任度** | 高信任度时，盼盼会主动优化工坊布局、提出升级建议。 |
| **情绪** | 心情好时制作品质有加成；疲惫时容易失败。 |
| **能量** | 制作消耗能量，需注意盼盼电量。 |
| **旅行技能** | 旅行可能带回新配方和特殊材料。 |

## 八、总结

工坊属性系统为星夜小馆增添了DIY的乐趣和温情细节。玩家通过盼盼在工坊中制作物品，既能提升小馆的硬件品质，又能加深与邻居的情感联系。工坊与后院、厨房的联动，让整个小馆的经营更加立体。随着工坊等级的提升，盼盼的创造力也得到发挥，最终可能制作出与祖父记忆相关的重要物品，影响游戏结局。
